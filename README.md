# gsw_passing_distribution

项目简介
我们评估了网络分析并且利用NBA.com丰富的的数据对金州勇士的传球网络进行可视化。
对数据进行分析来讨论为什么勇士队是冠军的最大热门

安装要求
1、安装Gephi用于可视化。
2、安装py文件中需要的各种库（可用pip进行安装较为简便）。

数据的获取
使用 playerdashptpass 的端点并且将同队所有球员数据保存到本地的 JSON 文件中。
数据来自 2015-16赛季的传球记录。

运行步骤
1、运行gsw_passing_network_analysis.py用于数据的分析
2、运行gsw_passing_network.gephi用于网络的可视化
